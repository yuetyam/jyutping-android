Android 粵語拼音輸入法。  
採用 [香港語言學學會粵語拼音方案](https://jyutping.org/jyutping) (粵拼 / Jyutping)，兼容各種習慣拼寫串法。  
候選詞會標注對應嘅粵拼。支援簡、繁體漢字。  
可以用倉頡、速成、筆畫、普通話拼音、拆字等反查粵語拼音。

另有 iOS、iPadOS 及 macOS 版: [yuetyam/jyutping](https://github.com/yuetyam/jyutping)
